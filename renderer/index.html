Historique & retour arrière

Bientôt : consulter les anciennes versions d'un site client et revenir en arrière en un clic si une modification a cassé quelque chose. (Nécessite un ajout côté plugin WordPress Pegasus : sauvegarde/restauration des versions.)

Clé d'équipe

Ta clé est installée dans ~/.pegasus/team-key. Pense à ajouter le plugin dans Claude Code : Personnaliser → Plugins → OrphicDev/pegasus.

Connecter un site existant

Le site tourne déjà sous WordPress. Trois étapes, sans FTP ni mot de passe à copier — le site apparaît ensuite tout seul dans le parc.

  1. Installer le plugin Pegasus sur le WordPress du client.
    wp-admin → Extensions → Ajouter → Téléverser une extension → choisir le zip → Installer → Activer.
  2. Connecter le site. Dans le wp-admin, menu Pegasus → bouton « Connecter ce site à Claude ». Le mot de passe est créé, chiffré et enregistré automatiquement.
  3. Vérifier. Le site apparaît dans le parc.

Prérequis : compte administrateur sur le WordPress du client, et site en HTTPS (requis par les mots de passe d'application WordPress).

Ce que fera Ploutos

Construire des devis sur mesure en assemblant vos prestations et vos tarifs : shooting, montage, campagne, retouche… On choisit les services et les quantités, le total se calcule tout seul.

Catalogue de prestations & tarifs

Titan

Accès développeur — réservé aux super admins.

Ce que fait Titan — en un clic

Titan prépare un espace de travail complet et prêt à coder :

  • installe les outils requis (git, GitHub CLI, Node — via Homebrew si besoin)
  • télécharge le code source de toutes les apps (Olympus, Pegasus, Zevs)
  • installe toutes les dépendances (npm install partout)
  • génère une carte du code pour Claude

Ton Claude Code peut alors scanner le code, détecter des problèmes, corriger et ajouter des fonctionnalités sur n'importe quelle app.
